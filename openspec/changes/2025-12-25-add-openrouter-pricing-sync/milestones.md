# Milestones

## M1 - Requirements & Acceptance
- Entry criteria: User confirms OpenRouter units, schedule, and model scope.
- Exit criteria: `requirements-analysis.md` + `acceptance-criteria.md` complete.
- Required artifacts: Requirements + acceptance docs.

## M2 - OpenSpec Proposal (if applicable)
- Entry criteria: M1 complete.
- Exit criteria: `proposal.md`, `tasks.md`, and spec deltas drafted.
- Required artifacts: OpenSpec proposal + delta spec.

## M3 - Unit Test Coverage
- Entry criteria: Proposal approved.
- Exit criteria: Pricing normalization unit tests added and passing.
- Required artifacts: Unit test file(s) + results.

## M4 - Regression & Integration
- Entry criteria: Core sync implementation complete.
- Exit criteria: Acceptance scripts pass and integration with usage endpoints verified.
- Required artifacts: Acceptance script outputs.

## M5 - Release & Monitoring
- Entry criteria: M4 complete.
- Exit criteria: Deployed edge function and scheduled workflow verified.
- Required artifacts: Deployment logs + manual trigger evidence.
